Update product images from Google Drive

The module supports updating images based on Google Drive links.

- Users can directly update by modifying the link in the image URL field.

- Users can use a .csv file to update in bulk.

Upload google drive url on product template form.

The images will be updated after we update the google drive image URL in the image URL field.

*********

We can use the "Import Product Image" function to update images for multiple products.
